## 🎮 조작 방법 (Controls)

| 현재 상태 | 키 입력 | 행동 및 기능 |
| :--- | :--- | :--- |
| **지상 (Ground)** | `W, A, S, D` | 로컬 캐릭터 기준 전/후/좌/우 지상 이동 |
| | `마우스 이동` | 시점 회전 (Yaw, Pitch 화면 뒤집힘 방지 적용) |
| | `Space Bar (1회)` | 정석 지상 점프 발동 |
| | `Left Shift (지속)` | 지상 대시 (스프린트 속도 1.5배 증가) |
| **공중 (Air)** | `Space Bar (더블 탭)` | **6자유도 공중 비행 모드(Flying) 진입** (0.25초 내 연타) |
| **비행 (Flying)** | `W, A, S, D` | 바라보는 카메라 기수 방향 기준 6축 자유 이동 |
| | `Space Bar (지속)` | 수직 고도 상승 (Up) |
| | `Left Shift (지속)` | 수직 고도 하강 (Down) |
| | `Q / E` | 6자유도 좌/우 몸통 기울임 회전 (Roll) |
| | `Space Bar (더블 탭)` | **6자유도 비행 모드 해제** (자유 낙하 상태로 복귀) |
